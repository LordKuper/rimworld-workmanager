Work priority management mod for RimWorld

## Description
This mod automates assigning work priorities to your pawns and, optionally, their daily work schedules.
For a full feature overview see [About/About.xml](About/About.xml).

**Dependencies:** requires [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077) on all versions; additionally requires [LordKuper.Common](https://steamcommunity.com/sharedfiles/filedetails/?id=3531352422) on RimWorld 1.6.
